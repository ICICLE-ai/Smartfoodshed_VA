# Smartfoodshed_VA
Visual analytics system built for smart food shed, especially for PPOD and Cold Chain data

## how to install 
git clone the repo

cd Smartfoodshed_VA/new_gui/frontend

npm install

## how to run 
### backend 
cd Smartfoodshed_VA/new_gui/backend
python app.py 

### frontend 
cd Smartfoodshed_VA/new_gui/frontend
npm run serve

open localhost:8080 in your broswer
